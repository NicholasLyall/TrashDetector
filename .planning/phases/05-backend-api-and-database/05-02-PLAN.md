**apps/api/services/event_service.py** -- Business logic for event creation (per D-04, D-05, D-08, D-10, D-12, D-13, D-14):

Implement these functions:
1. `apply_fallback(label: str, confidence: float) -> tuple[str, bool]`:
   - If `confidence < CONFIDENCE_THRESHOLD` (0.7), return `("trash", True)` per D-04, D-05
   - Otherwise return `(label, False)`
   - Import `CONFIDENCE_THRESHOLD` from `apps.api.constants`

2. `compute_impact(label: str) -> tuple[float, float]`:
   - Look up weight from `CATEGORY_WEIGHTS[label]`
   - If `label == "trash"`, return `(0.0, 0.0)` per D-14
   - Otherwise compute `co2 = weight * CO2_MULTIPLIERS[label]`
   - Return `(weight, co2)` -- `(waste_diverted_kg, co2_saved_kg)`
   - Import `CATEGORY_WEIGHTS`, `CO2_MULTIPLIERS` from `apps.api.constants`

3. `async def create_event(label: str, confidence: float, source_device_id: str, image: UploadFile) -> dict`:
   - Validate `label` is in `VALID_CATEGORIES`, raise `HTTPException(400)` if not
   - Call `apply_fallback(label, confidence)` to get `(final_label, fallback_used)`
   - Call `compute_impact(final_label)` to get `(waste_diverted_kg, co2_saved_kg)`
   - Generate `timestamp = datetime.now(timezone.utc)`
   - Generate `image_path = f"{timestamp.strftime('%Y%m%d_%H%M%S')}_{final_label}.jpg"` per D-10
   - Read image bytes: `image_bytes = await image.read()`
   - Upload to Supabase Storage: `supabase.storage.from_("sort-images").upload(path=image_path, file=image_bytes, file_options={"content-type": image.content_type or "image/jpeg"})` -- NOTE: lowercase "content-type" per Pitfall 1
   - Get public URL: `image_url = supabase.storage.from_("sort-images").get_public_url(image_path)`
   - Generate `event_id = str(uuid.uuid4())`
   - Insert into sort_events: `supabase.table("sort_events").insert({...}).execute()` with all fields:
     - `"id": event_id`
     - `"timestamp": timestamp.isoformat()`
     - `"label": final_label` (per D-03: label == routed_bin after fallback)
     - `"routed_bin": final_label` (per D-02, D-03)
     - `"confidence": confidence` (original confidence, not modified)
     - `"image_url": image_url`
     - `"waste_diverted_kg": waste_diverted_kg`
     - `"co2_saved_kg": co2_saved_kg`
     - `"fallback_used": fallback_used`
     - `"source_device_id": source_device_id`
   - Return `{"id": event_id, "message": "event created"}`
   - Import `supabase` from `apps.api.db`

**apps/api/services/metrics_service.py** -- Aggregate computation (per D-07, D-15):

1. `def compute_metrics(events: list[dict]) -> dict`:
   - If `not events`, return all-zero `MetricsResponse`-shaped dict
   - `total = len(events)`
   - `non_trash = sum(1 for e in events if e["label"] != "trash")` -- per D-15
   - `recycling_rate = round(non_trash / total, 2)` -- per D-15: non-trash / total
   - `co2_saved_kg = round(sum(e["co2_saved_kg"] for e in events), 2)`
   - `waste_diverted_kg = round(sum(e["waste_diverted_kg"] for e in events), 2)`
   - `avg_confidence = round(sum(e["confidence"] for e in events) / total, 2)`
   - `uncertain_count = sum(1 for e in events if e["confidence"] < 0.7)`
   - `uncertain_rate = round(uncertain_count / total, 2)`
   - `fallback_count = sum(1 for e in events if e["fallback_used"])`
   - `fallback_rate = round(fallback_count / total, 2)`
   - Return dict with all 7 fields

2. `def compute_breakdown(events: list[dict]) -> list[dict]`:
   - Group events by `label`, count occurrences
   - For each category: `{"label": cat, "count": count, "percentage": round(count / total, 2)}`
   - Return list sorted by count descending
   - Per D-07: at most 4 categories (the 4 from VALID_CATEGORIES)

**apps/api/routers/events.py** -- POST /events and GET /events (API-02, API-03):

```python
from fastapi import APIRouter, UploadFile, File, Form, HTTPException
from apps.api.schemas import EventCreateResponse, EventResponse
from apps.api.services.event_service import create_event as create_event_service
from apps.api.db import supabase

router = APIRouter(tags=["events"])

@router.post("/events", response_model=EventCreateResponse)
async def create_event(
    label: str = Form(...),
    confidence: float = Form(...),
    source_device_id: str = Form(...),
    image: UploadFile = File(...),
):
    result = await create_event_service(label, confidence, source_device_id, image)
    return EventCreateResponse(**result)

@router.get("/events", response_model=list[EventResponse])
def get_events(limit: int = 20):
    response = (
        supabase.table("sort_events")
        .select("*")
        .order("timestamp", desc=True)
        .limit(limit)
        .execute()
    )
    return response.data
```

**apps/api/routers/metrics.py** -- GET /metrics and GET /metrics/breakdown (API-04, API-05):

```python
from fastapi import APIRouter
from apps.api.schemas import MetricsResponse, BreakdownResponse
from apps.api.services.metrics_service import compute_metrics, compute_breakdown
from apps.api.db import supabase

router = APIRouter(tags=["metrics"])

@router.get("/metrics", response_model=MetricsResponse)
def get_metrics():
    response = supabase.table("sort_events").select("*").execute()
    return compute_metrics(response.data)

@router.get("/metrics/breakdown", response_model=BreakdownResponse)
def get_breakdown():
    response = supabase.table("sort_events").select("*").execute()
    categories = compute_breakdown(response.data)
    return BreakdownResponse(categories=categories)
```
